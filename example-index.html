MoonNet Icon Pack - Example Header

About This Example

This file demonstrates the recommended header layout and favicon setup for the MoonNet DevOps site. The logo is displayed at 80px height as specified.

Recommended Head Tags

<!-- SVG favicon (modern browsers) -->
<link rel="icon" type="image/svg+xml" href="icons/favicon.svg">

<!-- PNG favicons (fallback) -->
<link rel="icon" type="image/png" sizes="32x32" href="icons/generated/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="icons/generated/favicon-16x16.png">

<!-- ICO favicon (legacy browsers) -->
<link rel="shortcut icon" href="icons/generated/favicon.ico">

<!-- Apple Touch Icon (iOS home screen) -->
<link rel="apple-touch-icon" sizes="180x180" href="icons/generated/apple-touch-icon-180x180.png">

<!-- PWA Manifest -->
<link rel="manifest" href="site.webmanifest">

<!-- Theme color for mobile browser UI -->
<meta name="theme-color" content="#082a4b">

<!-- Microsoft Tile -->
<meta name="msapplication-TileColor" content="#082a4b">

Icon Files

  • icons/logo.svg — Full logo (280×80) for headers
  • icons/favicon.svg — Simplified icon for browser tabs
  • icons/apple-touch-icon.svg — Icon with navy background for iOS
  • icons/generated/*.png — Raster exports at various sizes
  • icons/generated/favicon.ico — Combined ICO file
  • site.webmanifest — PWA icon manifest

Notes

  • The apple-touch-icon PNG uses a filled navy background (#082a4b) for optimal iOS appearance.
  • Header logo is set to 80px height; adjust as needed (40px/60px options available on request).
  • Run ./icons/make_icons.sh to regenerate PNG/ICO assets from SVG masters.
